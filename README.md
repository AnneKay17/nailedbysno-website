# NailedBySno Website 💅

A modern and responsive landing page built for **NailedBySno**, a nail technician business specializing in custom nail sets, French tips, pedicures, and nail art.

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Elegant beauty-inspired UI
- Featured work showcase
- Masonry-style gallery
- Services section
- About section
- Contact and booking section
- Booking policy access
- Smooth scrolling navigation
- Sticky navigation bar

## 🛠️ Built With

- React
- CSS3
- Vite

## 📸 Sections

### Hero
A welcoming introduction with a featured nail set and clear call-to-action buttons.

### Featured Work
Highlights some of the client's most popular nail sets.

### Services
Displays available services including:

- Gel Overlays
- Acrylic Overlays
- Acrylic Tips
- French Tips
- Builder Gel
- Rubber Base
- Pedicures
- Nail Removals

### Gallery
A curated collection of nail sets and pedicures showcasing different styles and designs.

### About
Introduces the business and the nail technician behind the brand.

### Contact
Allows clients to easily get in touch and book appointments.

### Booking Policy
Clients can review important booking information before making an appointment.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/AnneKay17/nailedbysno-website.git
```

Navigate into the project:

```bash
cd nailedbysno-website
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
